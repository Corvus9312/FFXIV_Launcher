# FFXIV Launcher

此工具提供一個簡單的介面，協助你快速啟動 **FFXIV 官方啟動器（繁中服）**，並自動輸入帳號、密碼與 OTP 驗證碼（TOTP），減少每次手動輸入的流程。

## 功能

- **帳號管理**
  - 新增/儲存多組帳號密碼
  - 從本機設定檔讀取並在下拉選單快速切換
- **OTP（兩步驗證）**
  - 支援上傳 QR Code 圖片，自動解析 OTP 密鑰
  - 支援 Google Authenticator 匯出格式（`otpauth-migration://...`）
  - 支援標準 `otpauth://...`（含 `secret=`）格式
- **啟動與自動輸入**
  - 啟動官方啟動器後，自動定位視窗並依序輸入：信箱 / 密碼 / OTP
  - 不依賴 Tab 切換，改用相對座標點擊欄位（較穩定）
- **系統設定**
  - 設定官方啟動器 `FFXIV_Launcher.exe` 路徑

## 使用方式

1. 打開程式 → 進入「系統設定」設定官方啟動器路徑  
2. 進入「帳號設定」新增帳號/密碼  
3. 上傳 OTP 的 QR Code 圖片完成綁定（成功後可儲存）  
4. 回到主頁選擇帳號 → 點「啟動遊戲」自動完成啟動與輸入

## 設定檔位置

設定會存放在 Windows 使用者目錄下：

- `%APPDATA%\FFXIV_Custom_Launcher\config.json`

## 注意事項 / 已知限制

- **自動輸入**依賴 Windows 視窗與滑鼠鍵盤自動化；執行期間請避免移動滑鼠、切換焦點或遮住官方啟動器視窗，否則可能輸入到錯誤位置。
- 若官方啟動器 UI 改版、縮放比例（DPI）或視窗大小不同，可能需要微調欄位座標參數。
- 工具僅協助自動輸入與啟動流程，**不修改**遊戲檔案。

## 下載與執行

- 下載 Release 的 **onedir** 資料夾版
- 執行：`FFXIV_Launcher.exe`
